Представители владельцев облигаций

На первой странице показан реестр ПВО. При нажатии на карточку открывается всплывающее окно с количеством выпусков облигаций, учитываемых по таблице выпусков.

В расчёт включены только выпуски со статусом «Торгуется». Если один и тот же выпуск встречался у разных ПВО, сохранена только последняя запись по такому выпуску.
